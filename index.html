Sample size calculation for pre-post survey R code:

`PrePostSS <- function(P1,P2,Za,Zb,DE,NRR){ P=(P1+P2)/2; Q=1-P g1=Zasqrt(2PQ) g2=Zbsqrt(P1(1-P1)+P2(1-P2)) n1=(g1-g2)^2 n2=(P2-P1)^2 n=((n1/n2)DE)(1+NRR) return(n) }

P1 = Baseline prevalence

P2 = Endline prevalence

Za = Significance level

Zb = Power

DE = Design effect

NRR = Non-response rate

PrePostSS(0.50,0.70, 1.96, -0.842, 1, 0.00)`
